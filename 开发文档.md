## 项目架构

本项目采用模块化架构，分为 `core` 核心模块和 `features` 功能模块。

```
Voice-main/
├── app/                    # 主应用入口
├── core/                   # 核心模块
│   ├── common/            # 通用工具类
│   ├── data/              # 数据层（数据库、存储）
│   ├── playback/          # 播放引擎（ExoPlayer封装）
│   ├── scanner/           # 媒体文件扫描
│   ├── search/            # 搜索功能
│   ├── sleeptimer/        # 定时功能核心逻辑
│   └── ui/                # UI 组件和主题
├── features/               # 功能模块
│   ├── bookOverview/      # 书籍列表界面
│   ├── bookmark/          # 书签功能
│   ├── cover/             # 封面处理
│   ├── folderPicker/      # 文件夹选择
│   ├── playbackScreen/    # 播放界面
│   ├── settings/          # 设置界面
│   ├── sleepTimer/        # 定时界面
│   └── widget/            # 桌面小部件
└── plugins/                # Gradle 构建插件
```

## 模块说明

### Core 模块

| 模块 | 职责 |
|------|------|
| `core/common` | 通用工具类、扩展函数、常量定义 |
| `core/data` | Room 数据库、DataStore 偏好设置、数据仓库 |
| `core/playback` | ExoPlayer 封装、播放控制、后台服务 |
| `core/scanner` | 媒体文件扫描、元数据提取 |
| `core/search` | 书籍搜索、过滤、排序 |
| `core/sleeptimer` | 定时关闭逻辑（按时间/按章节） |
| `core/ui` | Compose 主题、通用 UI 组件 |

### Features 模块

| 模块 | 职责 |
|------|------|
| `features/bookOverview` | 书籍列表展示、分类 |
| `features/bookmark` | 书签增删改查 |
| `features/cover` | 封面图片加载、裁剪 |
| `features/folderPicker` | 文件夹选择、权限处理 |
| `features/playbackScreen` | 播放界面、控制逻辑 |
| `features/settings` | 设置项管理 |
| `features/sleepTimer` | 定时设置界面 |
| `features/widget` | 桌面小部件实现 |

## 开发环境搭建

### 系统要求

- Windows 10/11 或 macOS 或 Linux
- 至少 8GB 内存
- 至少 10GB 磁盘空间

### 安装依赖

1. **JDK 25**
   - 下载：[Temurin JDK 25](https://github.com/adoptium/temurin25-binaries/releases)
   - 解压到 `tools/jdk/`

2. **Gradle 9.6.1**
   - 下载：[Gradle 9.6.1](https://services.gradle.org/distributions/gradle-9.6.1-all.zip)
   - 解压到 `tools/gradle/`

3. **Android SDK**
   - 下载：[Command Line Tools](https://developer.android.com/studio#command-tools)
   - 解压到 `tools/android-sdk/`
   - 安装平台：`sdkmanager "platforms;android-37"`

### 生成签名密钥

首次编译前需要生成签名密钥，用于给 APK 签名：

```bash
# 进入项目目录
cd D:\TRAE_Project\Voice

# 生成签名密钥（将密码替换为您自己的密码）
tools\jdk\bin\keytool.exe -genkeypair -v -keystore Voice-main\signing\my-release-key.jks -alias jinn -keyalg RSA -keysize 2048 -validity 9125 -storepass 您的密码 -keypass 您的密码 -dname "CN=您的名字, OU=Development, O=Jinn, L=Beijing, ST=Beijing, C=CN"

# 创建签名配置文件
echo STORE_PASSWORD=您的密码 > Voice-main\signing\signing.properties
echo KEY_ALIAS=jinn >> Voice-main\signing\signing.properties
echo KEY_PASSWORD=您的密码 >> Voice-main\signing\signing.properties
```

**参数说明**：
| 参数 | 说明 |
|------|------|
| `-keystore` | 密钥库文件名 |
| `-alias` | 密钥别名 |
| `-keyalg` | 密钥算法（RSA） |
| `-keysize` | 密钥长度（2048位） |
| `-validity` | 有效期（9125天≈25年） |
| `-storepass` | 密钥库密码 |
| `-keypass` | 密钥密码 |

**重要提示**：
- 签名密钥已添加到 `.gitignore`，不会推送到 GitHub
- 请备份您的签名密钥和密码，丢失后将无法更新应用

## 核心功能实现

### 播放控制

播放控制通过 `PlayerController` 实现：

```kotlin
// 播放/暂停
player.playPause()

// 快进/快退
player.seekTo(positionMs)

// 设置播放速度
player.setSpeed(1.5f)

// 设置音量增益
player.setGain(Decibel(3f))
```

### 定时关闭

定时关闭通过 `SleepTimer` 实现：

```kotlin
// 按时间定时
sleepTimer.enable(SleepTimerMode.TimedWithDuration(30.minutes))

// 按章节定时
sleepTimer.enable(SleepTimerMode.EndOfChapter)

// 禁用定时
sleepTimer.disable()
```

### 跳过片头片尾

跳过片头片尾通过 `VoicePlayer` 实现：

```kotlin
// 片头跳过（章节开头自动跳转到指定位置）
applySkipStart()

// 片尾跳过（接近章节结尾时自动跳到下一集）
checkSkipEnd()
```

配置存储在 DataStore 中：

```kotlin
// 跳过片头时长（秒）
skipStartSecondsStore: DataStore<Int>

// 跳过片尾时长（秒）
skipEndSecondsStore: DataStore<Int>
```

### 定集关闭

定集关闭通过计算总时长并启用定时实现：

```kotlin
fun onAcceptSleepEpisodeCount(count: Int) {
    // 获取实时播放位置
    val liveState = player.livePlaybackState(bookId)
    val currentPosition = liveState?.positionMs ?: book.content.positionInChapter
    
    // 计算剩余时长
    val currentChapterDuration = book.chapters[currentChapterIndex].duration
    val remainingInCurrentChapter = currentChapterDuration - currentPosition
    
    // 累加后续章节时长
    var totalDurationMs = remainingInCurrentChapter
    for (i in 1 until count) {
        totalDurationMs += book.chapters[currentChapterIndex + i].duration
    }
    
    // 转为分钟并启用定时
    val totalMinutes = (totalDurationMs / 1000 / 60).toInt()
    sleepTimer.enable(SleepTimerMode.TimedWithDuration(totalMinutes.minutes))
}
```

## 常见问题

### 编译失败

**问题**：`java.nio.file.FileSystemException: ... 没有更多文件`

**解决**：清理 Gradle 缓存

```bash
rm -rf .gradle-home/caches
rm -rf build-temp
```

**问题**：`Found item String/xxx more than one time`

**解决**：检查 strings.xml 中是否有重复的字符串定义

### 运行时问题

**问题**：播放界面崩溃

**解决**：检查 `BookPlayViewModel` 中的状态管理是否正确

**问题**：定时功能不生效

**解决**：检查 `SleepTimer` 的状态流是否正常更新

### 调试技巧

1. 使用 `Logger.d()` 输出调试日志
2. 查看 `Logcat` 中的 `Voice` 标签日志
3. 使用 Android Studio 的 Layout Inspector 检查 UI
